CMS Lider

Wordpress es el CMS líder en internet. Basado en el lenguaje PHP y base de datos MySql, Wordpress se ha convertido en sinónimo de Blog y Sitio Web.

Millones de sitios

Hoy mas de 60 millones de sitios web y blogs utilizan Wordpress como sistema central de gestión de contenidos, imágenes, usuarios y publicaciones.

Open Source

Wordpress es libre y gratuito bajo licencia GPLv2 y puede ser adaptado, extendido o modificado libremente. Amplia documentación y plugins disponibles.

Establecido en el tiempo

Desde 2003, el crecimiento de Wordpress ha sido constante, de los 10 millones de sitios mas visitados del mundo, mas de un 25% utilizar Wordpress.

Cloud hosting

Instalamos tu aplicación en nuestros Servidores de alta disponibilidad. El datacenter donde esta alojado el servidor dispone de energía redundante, control de sobrecarga y recuperación de procesos en caliente, permitiendo una tasa de disponibilidad del 99,99 %.

Instalación y configuración

Cada aplicación dispone de especificaciones técnicas de instalación, las cuales seguimos rigurosamente e incluimos todas las sugerencias que cada desarrollador principal dispone, generando una instalación de alta prestación, segura y de gran confiabilidad. Lista para ser utilizada por el administrador y los nuevos usuarios del sistema.

Mantenimiento

El software vive en un constante ciclo de mejora y con mayor razón las aplicaciones web, principalmente por el entorno de cambios continuos que vive internet. Por tal razón los desarrolladores principales de las aplicaciones sacan parches de seguridad y o versiones que resuelven los llamados 'bugs' de las versiones principales. Nuestra solución incluye la actualización de tu aplicación con estos parches para que tu instalación este siempre con el código mas reciente y seguro.

Soporte

Las aplicaciones de uso masivo son altamente confiables y seguras. Sin embargo todo software es vulnerable a errores. Nuestra solución incluye la asistencia ante estas contingencias, de manera que tu aplicación vuelva a su estado normal y operativo en el menor tiempo posible.
